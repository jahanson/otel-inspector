# ADR-001: Plan OTEL Inspector as one project, not a new initiative

## Decision

Plan **OTEL Inspector and Dashboard MVP** as a single Linear project under the existing Mabel strategic umbrella unless observability becomes a multi-project objective.

## Rationale

The work has one named outcome and one MVP target date. Creating a separate initiative now would add hierarchy before the work earns it.

## Consequences

- Milestones carry the build phases.
- Issues carry implementation/evidence tasks.
- Follow-on traces/logs/proxy work can become new projects later if needed.
