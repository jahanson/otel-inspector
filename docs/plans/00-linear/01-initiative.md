# Initiative Recommendation

## Recommendation

Use the existing initiative by default:

```text
Mabel — Portable AI Work Partner
```

Do **not** create a new initiative called `OTEL Inspector`, `Observability`, or `Dashboard` yet. This is one project with a clear MVP outcome. The Linear Method warns against turning every workstream into roadmap hierarchy before the work has earned it.

## Attach this project

```text
Project: OTEL Inspector and Dashboard MVP
Target: 2026-08-14
```

## Promote to a new initiative only when

Create a future initiative such as `Mabel Runtime Observability` only if at least two serious projects share the same strategic objective, for example:

- OTEL Inspector and Dashboard MVP.
- Mabel CLI telemetry instrumentation.
- Cross-host runtime event standard.
- Hosted/team observability bridge.

Until then, extra hierarchy would be a very fancy sock drawer.
