# Milestone Close Checklist

- [ ] Issues are complete or intentionally moved.
- [ ] Acceptance criteria are met or explicitly descoped.
- [ ] Tests/fixtures/snapshots/evidence exist where required.
- [ ] Misses are reflected in status update or follow-on issue.
- [ ] Blockers reviewed and stale relations removed or downgraded to Related.
- [ ] Durable decisions promoted to ADRs/project-library docs if needed.
